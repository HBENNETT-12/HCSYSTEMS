Desk Setup Project Report

Overview

This desk setup was designed to support a wide range of tasks including software development, system administration, homelabbing, and content creation. The goal was to create a functional, scalable workspace that balances performance, comfort, and accessibility to surrounding infrastructure.

Core Workspace

At the center of the setup is a height-adjustable standing desk, allowing for both seated and standing work throughout the day. This improves ergonomics and supports long work sessions without sacrificing comfort.

Primary input devices include a Dell Premier keyboard and mouse, chosen for their reliability and clean, professional design.

Display Configuration

The workstation features a dual ultrawide monitor setup:

  • Samsung 34" Ultrawide
  • LG 34" Curved Ultrawide

This configuration provides an expansive workspace ideal for multitasking, virtualization management, and monitoring multiple systems simultaneously.

Audio & Peripherals

The setup includes a pair of desktop speakers for general audio use, along with a HyperX Cloud headset for focused work and communication.

For voice input and content creation, a Blue Yeti microphone mounted on a boom arm ensures high-quality audio while maintaining desk space and flexibility.

System Specifications

The primary workstation is built to handle demanding workloads such as virtualization, development, and gaming:

  • CPU: Intel i9-9900K
  • Memory: 32GB RAM
  • Storage: 4TB combined (SSD and HDD)
  • GPU: NVIDIA RTX 2080
  • Motherboard: B550 chipset

This configuration provides a strong balance between compute power, storage capacity, and graphical performance.

Extended Infrastructure

This desk setup is tightly integrated into a broader homelab environment:

  • Left Side: A dedicated server cabinet (detailed further in the Homelabbing Project) used for storage, virtualization, and network services.
  • Right Side:
    • Proxmox cluster for virtualization and distributed workloads
    • Ender 3 3D printer running OctoPi for remote printing and monitoring
    • Soldering equipment for hardware projects and repairs

This layout enables quick physical access to infrastructure while maintaining a clean and organized primary workspace.

Conclusion

This setup reflects a hybrid approach between a professional workstation and a personal lab environment. It is designed for flexibility, performance, and continuous expansion, supporting both current needs and future projects.
